Where My Bus App

Where My Bus is an Android application designed to help commuters locate nearby buses in real-time.  
The app aims to make public transport more convenient, reliable, and accessible for everyone.

---

Features

- Find Nearby Buses — View buses near your location using the Google Maps API.  
- Live Location Tracking — Real-time tracking of bus movement.  
- User Registration & Login — Secure authentication for passengers and drivers.  
- Interactive Map Interface — Smooth and intuitive map integration.  
- Custom UI Theme — Grey and yellow color scheme inspired by city buses.

---

Tech Stack

| Component | Technology |
|------------|-------------|
| **Language** | Kotlin |
| **IDE** | Android Studio |
| **Database** | Room Database / SQLite |
| **Maps API** | Google Maps Platform |
| **UI Design** | XML layouts + Material Design |
